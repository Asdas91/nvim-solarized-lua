## Installation

### VIM Packages
```
git submodule add --name nvim-solarized-lua https://github.com/ishan9299/nvim-solarized-lua
pack/packages/start/solarized
```

### Plug
```
Plug 'ishan9299/nvim-solarized-lua'
```
